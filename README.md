##  Présentation
Stud'Exchange est un projet de marketplace solidaire entre étudiants. Actuellement, le projet se concentre sur l'interface utilisateur et l'expérience de navigation.

##  Stack Actuelle

Frontend : HTML5, CSS3.

Design : Architecture responsive .

##  Évolutions à venir 
Le projet est en constante évolution. Les prochaines étapes techniques sont :

Backend : Intégration de Supabase pour dynamiser la plateforme.

Gestion de données : Création d'une base de données pour stocker les métadonnées des fiches de révision.

Utilisateurs : Mise en place d'un système de connexion (Login/Signup).

Stockage : Hébergement réel des fichiers PDF via un service de Cloud Storage.
